# 计算器

这是一个使用 Vue.js 构建的简易计算器 Web 应用程序。

## 功能

- 基本的算术运算：加法、减法、乘法、除法
- 对表达式进行求值
- 响应式布局

## 入门

要在本地运行该项目，请按照以下简单步骤操作。

### 先决条件

- 在本地计算机上安装 Node.js

### 安装

1. 克隆仓库
   ```sh
   git clone https://github.com/zhaozhiwei1992/calculator.git
2. 进入项目目录
   ```sh
   cd calculator
3. 安装依赖项
   ```sh
   npm install
4. 启动开发服务器
   ```sh
   npm run dev

### 使用方法

1. 使用按钮输入数字并进行计算
2. 按下“=”按钮对表达式进行求值
